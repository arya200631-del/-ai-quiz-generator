# AI Study Quiz Generator

A simple Python quiz application that generates questions based on a selected topic.

## Features

* Topic-based quizzes
* Score calculation
* Instant feedback
* Beginner-friendly Python project

## Topics Available

* Artificial Intelligence
* Python
* General Knowledge

## How to Run

1. Install Python
2. Download the project
3. Run the program

```
python quiz_generator.py
```
